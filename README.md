# 8086 emulator

All opcodes are supported.
Only the CPU itself is implemented, no controllers, peripheral or any other devices.
However, 80×25 monochrome text mode is supported by writing to the B0000 address block.

Used documentation:
- [Intel 8086 Family User's Manual October 1979](https://edge.edx.org/c4x/BITSPilani/EEE231/asset/8086_family_Users_Manual_1_.pdf)
